# 🔮 Mystical RAG: Simple PDF Chatbot

This is a Retrieval-Augmented Generation (RAG) web application built with Streamlit and LangChain. It allows users to upload any PDF document and instantly ask questions about it, using a local FAISS vector database to ensure accurate, hallucination-free answers.

## 🛠️ Built With
* **Python UI:** Streamlit
* **AI Brain:** OpenAI (`gpt-4o-mini`)
* **Framework:** LangChain (Using modern 2026 core imports)
* **Vector Store:** FAISS (Facebook AI Similarity Search)
* **Document Parsing:** PyPDF2 & RecursiveCharacterTextSplitter
